## Site CMS Detection Patterns

### Detection Strategy
Perform HTTP GET to the URL, then analyze:
1. Response headers
2. HTML content (meta tags, script srcs, CSS paths, comments)
3. URL patterns (wp-json, shopify CDN, etc.)

### Detection Signals Table

| CMS | Header Signals | HTML Signals | URL Signals | Confidence |
|---|---|---|---|---|
| WordPress | X-Powered-By: PHP, link: rel=https://api.w.org/ | /wp-content/, /wp-includes/, meta generator="WordPress" | /?p=, /wp-json/ | HIGH |
| Shopify | x-shopify-stage, powered-by: Shopify | Shopify.theme, cdn.shopify.com, "Shopify.theme" | .myshopify.com | HIGH |
| Wix | x-wix-request-id, X-Wix-Published-Version | static.wixstatic.com, "_wix_", parastorage.com | wixsite.com | HIGH |
| Tilda | — | tilda.cc/zero, tildacdn.com, "tilda" in scripts | tilda.cc | HIGH |
| GTM (has tag mgr) | — | GTM-XXXXX, googletagmanager.com/gtm.js, gtag/js | — | MEDIUM |
| Bitrix | x-powered-cms: Bitrix | /bitrix/, BX.ready | — | HIGH |
| Custom HTML | — | none of above, plain HTML | — | LOW |

### Python Detection Algorithm
```python
import requests, re
from urllib.parse import urlparse

def detect_cms(url, timeout=15):
    try:
        resp = requests.get(url, timeout=timeout, allow_redirects=True,
            headers={"User-Agent": "Mozilla/5.0 (compatible; ExtellaBot/1.0)"})
    except Exception as e:
        return {"cms_type": "unknown", "error": str(e), "confidence": 0.0}

    headers = {k.lower(): v.lower() for k, v in resp.headers.items()}
    html = resp.text.lower()
    signals = []
    scores = {}

    # WordPress
    if "/wp-content/" in html or "/wp-includes/" in html:
        scores["wordpress"] = scores.get("wordpress", 0) + 0.7
        signals.append("wp-content in HTML")
    if "wordpress" in html:
        scores["wordpress"] = scores.get("wordpress", 0) + 0.2
        signals.append("wordpress in HTML")
    if "api.w.org" in str(headers):
        scores["wordpress"] = scores.get("wordpress", 0) + 0.5
        signals.append("WP API header")

    # Shopify
    if "cdn.shopify.com" in html or "shopify.theme" in html:
        scores["shopify"] = scores.get("shopify", 0) + 0.8
        signals.append("Shopify CDN")
    if "powered-by" in headers and "shopify" in headers.get("powered-by",""):
        scores["shopify"] = scores.get("shopify", 0) + 0.9
        signals.append("powered-by: Shopify")

    # Wix
    if "static.wixstatic.com" in html or "parastorage.com" in html:
        scores["wix"] = scores.get("wix", 0) + 0.9
        signals.append("Wix static CDN")

    # Tilda
    if "tildacdn.com" in html or "tilda.cc" in html:
        scores["tilda"] = scores.get("tilda", 0) + 0.9
        signals.append("Tilda CDN")

    # Bitrix
    if "/bitrix/" in html:
        scores["bitrix"] = scores.get("bitrix", 0) + 0.8
        signals.append("Bitrix path")

    # GTM detection (regardless of CMS)
    has_gtm = bool(re.search(r"gtm-[a-z0-9]+", html, re.IGNORECASE))
    if has_gtm:
        signals.append("GTM tag found")

    if not scores:
        return {"cms_type": "unknown", "has_gtm": has_gtm,
                "confidence": 0.0, "signals": signals}

    best_cms = max(scores, key=scores.get)
    confidence = min(scores[best_cms], 1.0)
    return {"cms_type": best_cms, "has_gtm": has_gtm,
            "confidence": round(confidence, 2), "signals": signals}
```

### CMS → Install Method Mapping
| cms_type | Auto Install | Manual Fallback |
|---|---|---|
| wordpress | WP REST API + FTP to header.php | Paste in theme header.php |
| shopify | Shopify GraphQL scriptTagCreate | Shopify Admin > Themes > Edit code |
| gtm | GTM API create tag | GTM UI > New Tag |
| wix | Not available | Wix > Site > Custom Code |
| tilda | Not available | Tilda > Site Settings > HTML code |
| bitrix | FTP inject | Bitrix admin |
| custom | FTP inject | Direct HTML edit |
| unknown | FTP if credentials provided | Manual with code |
